# vpn_join
第三方节点接入，一键式启动，并接入去中心化Tenon VPN网络，提供服务和路由

当前仅支持： centos 7

# 接入步骤

1. 下载代码： 
  
   git clone https://github.com/actantion/vpn_join.git
   
   或者
   
   wget https://github.com/actantion/vpn_join/archive/1.1.1.zip
   

2. 执行： sh init_env.sh  
   执行完成后会重启系统
   
   
3. 重启完成，执行： sh start_node.sh
